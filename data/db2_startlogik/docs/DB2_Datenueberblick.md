# DB2 – Datenüberblick

## Eingaben
- DB1: Referenzdaten für Ligen, Namen, Nationalitäten, Charaktere, Spielertypen
- DB3: Vereinsdaten / Startvereine / Budgets / Prestigewerte
- DB4: Spielerstruktur, Positionsgewichte, Overall-Formeln
- DB5: Formationen, Taktikpositionen, Rasterlogik

## Hauptausgabe
Ein vollständiger `gameState` für:
- Vereinsauswahl
- Vereinsprofilansicht
- Büro
- Kader
- Aufstellung

## Kernbereiche im gameState
- `meta`
- `career`
- `world`
- `clubs`
- `players`
- `selection`
- `manager`
- `tactics`
- `ui`
