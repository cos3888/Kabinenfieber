# DB5 – Beschreibung für Menschen

DB5 ist die Taktikdatenbank. Sie erklärt, wie das Aufstellungsmenü fachlich funktioniert.

## Wofür sie gebraucht wird
- welche Positionen es gibt
- wo diese Positionen auf dem Spielfeldraster liegen
- welche Standardformationen verfügbar sind
- wie gut oder schlecht ein Spieler auf einem bestimmten Feldslot funktioniert

## Wichtig
DB5 macht die Taktik nicht nur optisch, sondern regelbasiert nutzbar.
